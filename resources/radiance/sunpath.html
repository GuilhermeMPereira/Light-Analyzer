Diagramas de trajetória solar

Esta página fornece diagramas de trajetória solar para cada 1° de latitude para hemisfério Norte e Sul. Os documentos são fornecidos em Portable Document Format (PDF), com os diagramas reais chegando em vetor formulário para impressões de alta qualidade. O tamanho da página é A4 com uma folha por arquivo. O tamanho médio do arquivo é de 90 kB.

Como os diagramas servem como material didático, algumas das anotações são deixados para o aluno, por exemplo, as horas e estações do caminho do sol.

O tempo é mostrado como hora solar local em vez de hora do relógio para fazer os diagramas menos confuso. Um gráfico da equação do tempo está incluído em cada documento.
