hdrgen

hdrgen é o mecanismo de composição HDR por trás do Photosphere e do WebHDR. Binários pré-compilados estão disponíveis para diversas plataformas. Como é o irmão maior Photosphere, as imagens são alinhadas automaticamente compensar o movimento da câmera.

Formatos

bracket

Bracket é um gerenciador de fotos HDR multiplataforma. Ele também se interessa por uma faixa de baixa dinâmica Gerenciador de fotos (LDR) com suporte aos formatos de imagem JPEG e TIFF padrão. bracket pode ser usado para criar, visualizar, analisar, calibrar, mapa de tons e exibir imagens HDR. Permite a conversão entre diferentes formatos de imagem, e suporta tarefas básicas de edição de imagens, como rotação, corte, inversão e redimensionamento.

Formatos

RGBE, LogLuv TIFF, TIFF de 48 bits, FP TIFF, OpenEXR

PFSTools

PFSTools é um conjunto de programas de linha de comando (e uma GUI) para leitura, escrita e manipulação e visualização de imagens e quadros de vídeo de alta faixa dinâmica (HDR). Todos os programas no troca de pacotes de dados usando um formato de arquivo genérico simples (pfs) para dados HDR. O O conceito do pfstools é semelhante ao pacote netpbm para imagens de baixa faixa dinâmica. pfstools são complementados por pfscalibration e pfstmo para criação de imagens HDR, e para operações de mapeamento de tons, respectivamente.

Formatos
