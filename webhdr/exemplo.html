Nesta etapa vamos apresentar uma sequencia de imagens e explicar de como é aplicado o HDR.

Formatos HDRI

High Dynamic Range

As imagens abaixo foram tiradas com uma Nikon Coolpix 995 câmera digital. O suporte de exposição integrado permite tire uma série de 5 imagens que podem ter até +/-2 f-stops sub ou superexposto.

Sequência de Richmond Park entre colchetes de exposição no sul de Londres

As imagens entre colchetes de exposição podem ser montadas em uma imagem de alta faixa dinâmica (HDR). Esta imagem tem muito maior faixa de valores de brilho que ele pode armazenar. O as imagens na animação abaixo são produzidas a partir do mesmo Imagem HDR no formato RGBe da Radiance. A exposição pode ser definida interativamente no visualizador ximage que faz parte do Conjunto de programas Radiance.

Visualizador HDR interativo do pfstools. Certifique-se de que o JavaScript esteja habilitado no seu navegador.

As câmeras digitais modernas armazenam informações adicionais no Arquivo JPEG que eles produzem. Isso inclui detalhes sobre o câmera, bem como valores de exposição e uma visualização em miniatura. As informações são mantidas no cabeçalho do arquivo e são armazenado em Formato EXIF.


Para determinar a exposição de uma imagem, as seguintes informações são necessárias:


  • • Velocidade do filme ISO
  • • Abertura
  • • Tempo de exposição

Todas essas informações são armazenadas no cabeçalho EXIF. Portanto, é possível utilizar uma imagem HDR, criada a partir de uma série de imagens com variação de exposição, para determinar a distribuição real de luminância dentro da cena fotografada, embora isso não seja tão preciso quanto usar um medidor de luminância calibrado ou uma (muito cara) câmera CCD calibrada.


Exposição = log2 ( Abertura2 * (1/Velocidade do obturador) * (Velocidade ISO/100) )

Com esta informação, podem ser criadas imagens em cores falsas que mapear a luminância de um pixel para um pixel escolhido arbitrariamente cor. Normalmente, os tons azuis são escolhidos para áreas escuras, enquanto tons vermelhos indicavam altas luminâncias.


Outra aplicação das informações HDR contidas no a imagem serve para simular como a cena apareceria para um espectador humano. Radiance tem o pcond comando para isso.

Se você quiser tentar ver para que pode usar uma imagem HDR, sinta-se à vontade para baixar este é o Richmond Park, no sul de Londres (2,4 MB), que é a partir do qual os exemplos acima são produzidos.
